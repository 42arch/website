### Rust

Rust's strict type system and borrowing rules look great with the new theme:

```rust
fn main() {
    let mut system_status = String::from("Booting");
    
    // Ownership transferred and borrowed
    print_status(&system_status);
    
    system_status.push_str("... Done!");
    println!("Final Status: {}", system_status);
}

fn print_status(status: &str) {
    println!("Current System Status: {}", status);
}
```

### Python

A quick snippet of Python to demonstrate standard library imports and list comprehensions:

```python
import os
import json

def load_config(path: str) -> dict:
    """Loads a JSON configuration file."""
    if not os.path.exists(path):
        raise FileNotFoundError(f"Config file not found at: {path}")
        
    with open(path, 'r') as file:
        data = json.load(file)
        
    # Process valid keys
    return { k: v for k, v in data.items() if v is not None }

config = load_config('./os-config.json')
print(config)
```
