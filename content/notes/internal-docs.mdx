# Architecture Notes

The `folio-os` project is built with:

1. **Next.js 16** for the framework.
2. **Tailwind CSS 4** for styling.
3. **Fumadocs** for content management.
4. **Zustand** for workspace state.

## Core Components

- `WorkspaceShell`: The main container.
- `ExplorerSidebar`: Content navigation.
- `TabBar`: Active document management.
