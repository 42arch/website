## 1 角度和弧度

### 1.1 角度与弧度的定义

- 角度 (Degree)：角度是最常见的角度表示方式，单位为度（°）。圆周被分为360°，所以一圈360°。

- 弧度（Radian）：弧度是另一个常见的角度表示方式，单位为弧度（rad）。角度θ（以弧度为单位）是当圆弧长度等于圆的半径时形成的角度。
  若圆弧长度等于半径 (r)，则 θ 为1弧度。一个完整圆的弧度是 $2\pi$。

$$
2\pi \text{ rad} = 360^\circ
$$

### 1.2 角度与弧度互转

通过以下公式进行转换：

以度数表示的角，把数字乘以 $\frac{\pi}{180}$ 便转换成弧度。

$$
\text{弧度} = \text{角度} \times \frac{\pi}{180}
$$

以弧度表示的角，乘以 $\frac{180}{\pi}$ 便转换成角度。

$$
\text{角度} = \text{弧度} \times \frac{180}{\pi}
$$

即 1 弧度约为57.3°，计算方式：$ 1 \text{弧度} = \frac{180}{\pi}\text{度} = \frac{180}{3.14159} \approx 57.2958^\circ $

### 1.3 弧度制的使用场景

在三角函数的计算中，如`Math.sin(x)`中的x参数是弧度。在极坐标系中，角度通常以弧度为单位表示。

### 1.4 代码示例

```js
const degrees = 30 // 角度
const radians = degrees * (Math.PI / 180) // 转换为弧度
const sinValue = Math.sin(radians) // 计算正弦值
console.log(sinValue) // 输出 0.5
```

## 2 三角函数

### 2.1 定义

三角函数是数学中描述角度与三角形边长关系的函数。
它们基于直角三角形的基本特性，定义了角度与边长之间的关系，并推广到单位圆和**周期性函数**。

### 2.2 基本三角函数

(1) 正弦函数 $\sin$

$$
\sin(\theta) = \frac{\text{对边}}{\text{斜边}}
$$

- 定义：直角三角形中，角 $\theta$ 的对边长度与斜边长度的比值。
- 取值范围：$-1 \leq \sin(\theta) \leq 1$

(2) 余弦函数 $\cos$

$$
\cos(\theta) = \frac{\text{邻边}}{\text{斜边}}
$$

- 定义：直角三角形中，角 $\theta$ 的邻边长度与斜边长度的比值。
- 取值范围：$-1 \leq \cos(\theta) \leq 1$

(3) 正切函数 $\tan$

$$
\tan(\theta) = \frac{\text{对边}}{\text{邻边}} = \frac{\sin(\theta)}{\cos(\theta)}
$$

- 定义：直角三角形中，角 $\theta$ 的对边长度与邻边长度的比值。
- 取值范围：$-\infty < \tan(\theta) < \infty$
- 特性：当 $\cos(\theta) = 0$ 时，正切函数无定义（例如 $\theta = 90^\circ$ 时）。
