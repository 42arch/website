## 目标

一个像素游戏风格的个人网站，采用像素字体和复古游戏美学，展示个人品牌、博客文章、项目作品集，以及个人介绍。目标是让访客在浏览时获得沉浸式像素游戏体验，同时清晰了解博主的身份与作品。

页面包括：首页、项目展示、博客文章、关于我等。

## 主题

网站的域名为：starllow.com。starllow 为单词 Star 和 Swallow 的结合。网站的名称为 Starllow Lab，以一个实验室为主题，成员主要是博主（42arch）。网站吉祥物是一个卡通星星形象。

## 核心流程

These flows must work end-to-end:

访客打开网站 → 看到像素风首页 Hero（含打字机动画欢迎语、像素角色/头像）→ 导航至各页面

访客点击「博客」→ 看到博客文章列表（标题、标签、日期）→ 点击文章 → 进入文章详情页阅读全文

访客点击「项目」→ 看到项目卡片网格（项目名、描述、技术标签、链接）→ 了解项目详情

访客点击「关于我」→ 看到像素风个人介绍（头像、技能像素进度条、个人故事）

## 不要做的事

❌ 不使用真实后端，博客和项目数据均为静态 mock 数据

❌ 不使用圆角卡片/扁平设计，保持像素风硬边框风格

❌ 不使用渐变柔和色调，保持高对比像素游戏色彩

## 技术要求

Next.js + TypeScript + Tailwind CSS + Shadcn/UI。使用 Google Fonts 的 Press Start 2P 像素字体。博客文章和项目数据使用本地静态数据（mock data）存储在前端。路由使用客户端路由实现多页面切换。

## 设计偏好

像素游戏风（Pixel Art / Retro Game）美学：

背景：深色系（深蓝/深绿/黑色）带像素网格纹理

字体：像素字体（Press Start 2P from Google Fonts）

色彩：高对比亮色点缀（亮绿、亮黄、亮青、品红），类似经典像素游戏调色板

边框与卡片：像素风8-bit样式边框（双线/阴影偏移）

动效：打字机效果、闪烁光标、像素溶解过渡、8bit风格hover效果

导航：顶部导航栏带像素风选中指示器（▶ 光标）

整体感：如同在玩一款RPG游戏的主菜单界面
